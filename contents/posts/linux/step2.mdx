### 階層とは

例えば下記のような構成のディレクトリがあったとします。

![Photo by Minh Pham](/images/documents/linux/ディレクトリ.png)

**階層**とはディレクトリの保存されている場所の階層のことをいいます。
ファイルを保存する場所は**ディレクトリ**という単位で分けられています。

***

#### 階層の関係

![Photo by Minh Pham](/images/documents/linux/second-explain.png)

例えば上記図で言うと
第2階層の親階層は第1階層になります。

第2階層を基準として第1階層と第2階層の関係は
> 親階層→第1階層
> 子階層→第3階層

となります。

### 絶対パス/相対パス

ファイルの指定の基本相対パスと絶対パスを紹介します

<InfoBox type="info" title="パスとは">
ファイルを開いたりコマンドを実行したりするためのファイルやコマンドの場所を指定するための指定方法になります。
コンピュータ上のファイルやディレクトリの「住所」のようなものです。
</InfoBox>

<Note type="tip">
**日常生活で例えると...**
パスは住所のようなものです。友達に「家に来て」と伝える時に、
- 「東京都渋谷区〇〇1-2-3」と伝えるのが**絶対パス**（どこからでも辿り着ける）
- 「駅から2つ目の角を右」と伝えるのが**相対パス**（今いる場所によって道順が変わる）

このような違いがあります。
</Note>

![Photo by Minh Pham](/images/documents/linux/絶対パス.png)

#### 絶対パスとは

![Photo by Minh Pham](/images/documents/linux/絶対パス.png)

ルートディレクトリを基準点として指定する方法です。
この表示方法は、カレントディレクトリ（現在表示しているディレクトリ）がどこであっても、
間違いなく目的のファイルを指定できます。

<InfoBox type="info" title="用語解説：ルートディレクトリ">
**ルートディレクトリ**とは、対象のディレクトリがある最上位のディレクトリのことを指します。
Linuxでは「/」（スラッシュ）で表され、すべてのディレクトリの起点となる場所です。
木の根（root）のように、すべての枝（ディレクトリ）が派生する大元と考えるとわかりやすいでしょう。
</InfoBox>

<Note type="info">
**絶対パスの例：**
```
/Users/yonekura/Documents/practice/sample.txt
```
このパスは、どのディレクトリにいても必ず同じファイルを指し示します。
先頭の「/」がルートディレクトリを表しています。
</Note>

#### 相対パスとは

![Photo by Minh Pham](/images/documents/linux/相対パス.png)

カレントディレクトリを基準点として指定する方法になります。
つまり、ある対象のディレクトリからどういったパスでファイルを指定すれば
アクセスしたいファイルやディレクトリにアクセスできるかを示す方法です。

<InfoBox type="info" title="用語解説：カレントディレクトリ">
**カレントディレクトリ**とは、現在自分がいるディレクトリのことです。
「current（現在の）」という意味で、作業中のディレクトリを指します。
pwdコマンドを使うと、カレントディレクトリの場所を確認できます。
</InfoBox>

<Note type="info">
**相対パスの例：**
もし現在のディレクトリが `/Users/yonekura/Documents` だとすると、
```
practice/sample.txt
```
と指定すれば、`/Users/yonekura/Documents/practice/sample.txt` にアクセスできます。

一つ上の階層に戻りたい場合は「..」（ドット2つ）を使います：
```
../Downloads/file.txt
```
</Note>

<Note type="warn">
**相対パスの注意点**
相対パスは現在いる場所によって意味が変わります。
カレントディレクトリが異なると、同じ相対パスでも違うファイルを指してしまうことがあるので注意しましょう。
</Note>

### Linuxの基本的なコマンド

#### pwdコマンド

```
yonekuradaisuke@yonekuradaisukenoMacBook-Pro-2 ~ % pwd
/Users/yonekuradaisuke
```

pwdコマンドはルートディレクトリから今いる階層までを絶対パスで表示してくれます

#### lsコマンド

```
yonekuradaisuke@yonekuradaisukenoMacBook-Pro-2 ~ % ls
Applications		Music			sample.txt
Creative Cloud Files	Pictures		sample1
Desktop			Postman			unimatrix
Documents		Public			www
Downloads		Sites			zshrc
Library			go
Movies			kouda
```

lsコマンドは今いる階層より一つ下にある階層のファイルや
ディレクトリを表示してくれるコマンドになります。

#### cdコマンド

```
$ cd  sample2
```

cdコマンドは階層を移動したい時に使用することができます。

#### mkdirコマンド

```
$ mkdir sample4
```

mkdirコマンドはmake direcrotyの略です。
mkdir ディレクトリ名
でディレクトリを作成することのできるコマンドです。

#### rmdirコマンド

```
$ rmdir sample2
```

空のディレクトリを削除できるコマンドです。

#### rmコマンド

```
$ rm sample3
```

rmコマンド(removefile)コマンドは空のファイルやディレクトリを
削除できるコマンドになります。

```
$ rm -r sample3
```

ちなみに、rmの右隣に半角スペース打って -r と入力することで
中身のあるディレクトリやファイルも削除することができます。
この-rをオプションコマンドと呼びます。

#### catコマンド

```
$ cat sample2.txt
```

catコマンドはファイルの中身を閲覧できるコマンドになります。
