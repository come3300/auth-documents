#### CUI操作を基本としている

Linuxは、テキストベースのコマンドラインインターフェース（CUI）を基本として操作されます。
これは、ユーザーがコマンドを入力してターミナル上で操作を行うことを意味します。

例えばCUIを使ってsampleという名のディレクトリを作成したい時には
下記のコマンドを入力します。

```bash
$ mkdir sample
```

このように入力することでsampleディレクトリが作成されます。

<Note type="tip">
**実際に試してみよう！**
Linuxのコマンドは実際に操作してみることで理解が深まります。まずは簡単なコマンドから試してみましょう。
</Note>

#### 実践：基本コマンドで操作を体験しよう

実際にコマンドを使って、Linuxの操作感覚を掴んでいきましょう。
以下の手順を順番に実行してみてください。

**ステップ1：現在の場所を確認する**

```bash
$ pwd
```

<InfoBox type="info" title="pwdコマンドとは">
**pwd**は「Print Working Directory」の略で、現在いるディレクトリ（フォルダ）の場所を表示してくれます。
自分が今どこにいるのかを確認するための基本的なコマンドです。
</InfoBox>

**ステップ2：練習用のディレクトリを作成する**

```bash
$ mkdir practice
```

**ステップ3：作成されたか確認する**

```bash
$ ls
```

<InfoBox type="info" title="lsコマンドとは">
**ls**は「list」の略で、現在のディレクトリ内にあるファイルやディレクトリの一覧を表示します。
作成した「practice」ディレクトリが表示されていれば成功です。
</InfoBox>

**ステップ4：作成したディレクトリに移動する**

```bash
$ cd practice
```

<InfoBox type="info" title="cdコマンドとは">
**cd**は「Change Directory」の略で、ディレクトリを移動するためのコマンドです。
「cd practice」と入力すると、practiceディレクトリの中に移動できます。
</InfoBox>

**ステップ5：移動できたか確認する**

```bash
$ pwd
```

実行すると、パスの最後に「/practice」が追加されていることが確認できます。
これで、practiceディレクトリに移動できたことがわかります。

<Note type="tip">
コマンド操作に慣れてきましたか？最初は戸惑うかもしれませんが、この4つのコマンド（pwd、mkdir、ls、cd）を覚えるだけで、Linuxの基本的な操作ができるようになります。
</Note>

### 無料で利用できる

Linuxは基本的に無料で利用可能です。

例えば、Windows11のOSを入れているPCでLinuxを使用したいときは
下記コマンドを入力します。

```bash
$ wsl --install
```

このコマンド一つでLinuxの導入が無料でできてしまいます。

他にも中古のPCを購入してその中にLinuxを導入してサーバーとして
利用するなんて方法もできてしまいます。
