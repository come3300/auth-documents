# 🗒️ memosテーブルを作成しよう

このパートでは、**メモ投稿機能のデータを保存するための `memos` テーブル** を MySQL に作成します。

---

## 🎯 目標

- `simple_memo` データベース内に **`memos` テーブル** を作成
- phpMyAdmin で **`users` テーブルと `memos` テーブル** が存在している状態にする

---

## 🛠️ 作業の流れ

1. phpMyAdmin にアクセス（`http://127.0.0.1:8081/`）
2. 左メニューから **`simple_memo` データベース** を選択
3. `SQL` タブをクリック
4. `memos` テーブル作成用の SQL を貼り付けて実行
5. 作成結果を確認

---

## 🖥️ 手順詳細

### 1. phpMyAdmin にアクセス

ブラウザで以下にアクセスします。

```
http://127.0.0.1:8081/

```

---

### 2. データベースを選択

左メニューから **`simple_memo`** をクリックします。

![データベース選択](/images/documents/php-memo/step13/select-database.png)

---

### 3. SQLタブを開く

データベースのメニューから **「SQL」** タブをクリックします。

---

### 4. SQLを貼り付けて実行

以下の SQL を貼り付け、**「実行」** ボタンをクリックします。

```sql
CREATE TABLE memos (
    id BIGINT NOT NULL AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    title VARCHAR(255),
    content TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
```

![SQL実行](/images/documents/php-memo/step13/sql-tab.png)

---

### 5. 結果を確認

実行が成功すると、左のテーブル一覧に **`memos`** が追加されます。

これで、`users` と `memos` の2つのテーブルが揃いました。

![テーブル確認](/images/documents/php-memo/step13/table-created.png)

---

## 📝 補足：テーブル構造の意味

| カラム名 | 型 | 説明 |
| --- | --- | --- |
| `id` | BIGINT | メモの一意ID（自動採番） |
| `user_id` | BIGINT | 投稿者ユーザーのID |
| `title` | VARCHAR(255) | メモのタイトル |
| `content` | TEXT | メモの本文 |
| `created_at` | DATETIME | 作成日時（自動設定） |
| `updated_at` | DATETIME | 更新日時（自動設定） |

---

- これで、ユーザーごとのメモ管理が可能になります。
