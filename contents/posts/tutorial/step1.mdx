## **JavaScriptとは**

**JavaScript**は、1995年にNetScapeが開発したスクリプト言語です。現在※でも様々な場所で使われており、
特にウェブ開発においては欠かせない存在となっています。
※2025年10月時点

### なぜJavaScriptが重要なのか？
ウェブサイトに動きと機能を与えるプログラミング言語です。
例として、自動販売機で考えてみましょう！

例えば：

### JavaScriptとJavaは別物

<InfoBox>
**よくある勘違い：JavaScriptとJavaは全く別物！**

名前が似ていますが、JavaScriptとJavaは全く異なる言語です。
よく初学者さんがJavaScriptのことをJavaと略すことがありますが、間違いです。
（私も最初は間違えてました笑）

**正しい略し方：**
- JavaScript → **JS** （これが正解）
- JavaScript → Java （これは間違い！）

**例えるなら：**
- ハムスター と ハム（肉）くらい違う
- インド と インドネシアくらい違う

全く別の言語なので、間違えないように気をつけましょう。
JavaScriptを略して表現するなら**JS**が一般的です。
</InfoBox>

## JavaScriptの特徴

JavaScriptの特徴として下記が挙げられます。

- 動的型付け言語
- クライアントサーバとサーバサイドの実装が可能

### 動的型付け言語
JavaScriptは、動的型付け言語の一つです。
動的型付け言語とは、プログラミングの実行時に代入された値などによって型を決定する言語です。

Javaは静的型付け言語に属するプログラミング言語です。動的型付け言語とは異なり、型をプログラムを実行する前に定める言語です。

動的型付け言語は「型」を気にする必要がないというメリットがありますが、バグが混入しやすいというデメリットもあります。
例えば、数字しか入れる予定のない変数に文字列が入ってしまう可能性があります。

### クライアントサイドとサーバーサイドの実装が可能
JavaScriptだけで、Webサービスを完結させることもできます。
プログラミングにおいて、目に見える部分の処理を書く「クライアントサイド」と目に見えない部分の処理を書く「サーバーサイド」に分けることがあります。
一般的には、クライアントサイドはJavaScriptで、サーバーサイドはJavaやPHPなどでWebサービスを立ち上げますが、Node.jsという実行環境を使えばJavaScriptだけで両方の役割を担うことが可能です。

## **JavaScriptを学習するメリット**
では、JavaScriptを学習するメリットとは具体的に何があるでしょうか？

### 1. 初心者におすすめ
JavaScriptは初心者におすすめのプログラミング言語です。
JavaScriptを動かすために必要な準備は少なく。Web上ですぐに出力結果を確認できます。
他の言語に比べて。書き方が簡単な点もおすすめと言えます

### 2. できることが多い
JavaScriptでできることは他の言語と比べても多いです。
例えば、Web上で動くゲームやアニメーションを作ることや
クライアントサイト、サーバーサイド両方で動かすことも可能です。

### 3. 情報量が多い
JavaScriptを学ぶ人は非常に多いため情報が多く、不明点があった場合に解決しやすいです。
調べたいキーワードを入力するだけでたくさんの情報を入手できます。

## **実際に動かしてみよう！**

### 方法1：ブラウザのコンソール（一番簡単）
1. ブラウザ（Chrome推奨）を開く
2. F12キーを押す（または右クリック→検証）
3. 「Console」タブをクリック
4. 以下のコードを入力してEnterキーを押す

```javascript
console.log("Hello World!");
```

### 方法2：HTMLファイルを作って実行（推奨）

実際の開発に近い形でJavaScriptを動かしてみましょう。

#### 1. HTMLファイルを作成する

テキストエディタ（メモ帳やVSCodeなど）を開いて、以下のコードをコピーして貼り付けます。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>はじめてのJavaScript</title>
    <style>
        body {
            font-family: sans-serif;
            padding: 20px;
            text-align: center;
        }
        button {
            font-size: 18px;
            padding: 10px 20px;
            margin: 10px;
            cursor: pointer;
        }
        #message {
            font-size: 24px;
            margin: 20px;
            color: #333;
        }
    </style>
</head>
<body>
    <h1>JavaScriptを体験しよう！</h1>
    <div id="message">ボタンを押してみてください</div>
    <button onclick="showMessage()">メッセージを表示</button>

    <script>
        function showMessage() {
            document.getElementById('message').textContent = 'こんにちは、JavaScript！';
            alert('JavaScriptが動いています！');
        }
    </script>
</body>
</html>
```

#### 2. ファイルを保存する

- ファイル名：`index.html`
- 保存場所：デスクトップやドキュメントフォルダなど、わかりやすい場所

<InfoBox>
**保存のポイント**

- 拡張子は必ず `.html` にする
- ファイル名は半角英数字で（日本語は避ける）
- 保存したフォルダの場所を覚えておく
</InfoBox>

#### 3. ブラウザで開く

保存した`index.html`ファイルをダブルクリックするか、
ブラウザにドラッグ&ドロップすると、ブラウザで開けます。

#### 4. ボタンを押してみる

「メッセージを表示」ボタンを押すと、
- 画面のテキストが変わる
- ポップアップ（アラート）が表示される

これがJavaScriptの動きです！

## 実践課題：自分で改造してみよう！

先ほど作ったHTMLファイルを改造して、JavaScriptの動きを体験しましょう。

### 課題1：クリックカウンター

ボタンを押した回数を数えるカウンターを作ってみましょう。

<InfoBox>
**ヒント：以下のコードに置き換えてみよう**

`<script>`タグの中身を以下に書き換えてみてください。

```javascript
let count = 0;  // カウントを記録する変数

function showMessage() {
    count = count + 1;  // カウントを1増やす
    document.getElementById('message').textContent =
        'ボタンが押された回数: ' + count + '回';
}
```

保存してブラウザを更新（F5キー）すると、
ボタンを押すたびに数字が増えていきます！
</InfoBox>

### 課題2：背景色を変更する

ボタンを押すと背景色が変わるようにしてみましょう。

<InfoBox>
**ヒント：背景色を変更するコード**

ボタンのHTMLを追加：
```html
<button onclick="changeColor()">背景色を変える</button>
```

`<script>`タグの中に関数を追加：
```javascript
function changeColor() {
    // ランダムな色を生成
    let colors = ['#FFB6C1', '#87CEEB', '#98FB98', '#FFD700', '#DDA0DD'];
    let randomColor = colors[Math.floor(Math.random() * colors.length)];
    document.body.style.backgroundColor = randomColor;
}
```

何度も押すと色が変わります！
</InfoBox>

### 課題3：時刻を表示する

現在の時刻を表示するボタンを作ってみましょう。

<InfoBox>
**ヒント：時刻を表示するコード**

```javascript
function showTime() {
    let now = new Date();
    let hours = now.getHours();
    let minutes = now.getMinutes();
    let seconds = now.getSeconds();

    document.getElementById('message').textContent =
        '現在の時刻: ' + hours + '時' + minutes + '分' + seconds + '秒';
}
```

ボタンに `onclick="showTime()"` を設定してみましょう。
</InfoBox>

<Note>
**うまく動かないときは**

- コンソールを開いてエラーメッセージを確認する（赤い文字で表示される）
- スペルミスがないか確認する（JavaScriptは大文字小文字を区別します）
- セミコロン（`;`）や括弧（`()`、`{}`）の閉じ忘れがないか確認する
- ブラウザを更新（F5キー）してみる

わからないことがあったら、エラーメッセージをコピーして検索してみましょう。
</Note>
